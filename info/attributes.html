Attributes

List of attributes that can be used on HTML elements.

Global Attributes

  • class
    The CSS or JS class of the element.
  • id
    Unique name for element.
  • contenteditable
    Allows content to be changed by the user.
  • lang
    Language of the content.
  • dir
    Direction the text flows.
  • charset
    The charset being used.

ARIA Roles

Accessibility attributes

  • aria-label
    The name of the element.
  • aria-hidden
    If the element will be seen.
