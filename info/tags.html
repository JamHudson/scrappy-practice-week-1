HTML Tags

Tags I didn't want to include but I still need to list technically.

  • iframe
    Allows embedding a video, sound, or other file from a different service.
  • div
    Block-level element with nothing.
  • span
    In-line element with nothing.
  • link
    Link to another file, like a stylesheet.
  • script
    Loads a javascript file.
  • footer
    Information that is additional to the content and not primary.
  • article
    "Wrapped around any instance of an article." A "unit of content"
  • section
    A new section of content.
  • aside
    Something off to the side, not necessarily related to the content.
  • main
    The main content of a page/article/section.
